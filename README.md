# temporal-skills

Claude Code skills for AI-assisted design and development of Temporal workflows.

## Current Skills

- **[design](./design/SKILL.md)** - Design Temporal workflows using a DSL with parser/LSP tooling and visualization (VSCode extension)

## Planned Skills

- **Implementers** - Generate workflow code from DSL designs (one skill per Temporal SDK: Go, TypeScript, Python, Java, etc.)
- **Translators** - Analyze existing systems (event-based architectures, SQS/Lambda, etc.) and generate equivalent DSL designs
- **Debuggers & Optimizers** - Assist with debugging, profiling, and optimizing existing Temporal workflows
