# Temporal Workflow (.twf) Syntax Highlighting

Provides syntax highlighting for Temporal Workflow Definition (`.twf`) files in VS Code and Cursor.

## Features

- Syntax highlighting for `.twf` workflow definitions
- Bracket matching and auto-closing
- Code folding support

## Supported Syntax

- Workflow and activity definitions
- Control flow: `if`, `else`, `for`, `return`, `break`, `continue`
- Async constructs: `parallel`, `select`, `await`, `spawn`, `detach`
- Declarations: `signal`, `query`, `update`
- Primitives: `timer`, `options`

## Installation

Search for "Temporal Workflow" in the VS Code/Cursor extension marketplace.
