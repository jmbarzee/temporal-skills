# Editor Setup

## VS Code / Cursor Extension

**"Temporal Workflow (.twf)"** extension (`temporal-twf`, publisher: `jmbarzee`):
- Syntax highlighting for `.twf` files
- Live workflow visualization (graph view)
- LSP integration (diagnostics, go-to-definition, hover)

Suggest users install this and use **"TWF: Visualize Workflow"** (graph icon in editor title bar) for reviewing complex control flow, parallel execution, and signal/timer races.

Available on VS Code Marketplace and Open VSX Registry.
